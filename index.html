Use on iPhone

iPhoneで3表ラボを使う

近くで使う

同じWi-Fi
  1. MacとiPhoneを同じWi-Fiにつなぎます。
  2. Macで start-iphone.command を開きます。
  3. Terminalに出た http://192.168.x.x:4173/ をiPhoneのSafariで開きます。
  4. Safariの共有ボタンから ホーム画面に追加 を選ぶと、アプリのように起動できます。

外出先で使う

HTTPS公開
  1. このフォルダをGitHub Pages、Netlify、Vercelなどに公開します。
  2. 公開された https://... のURLをiPhoneのSafariで開きます。
  3. 一度読み込むと、対応ブラウザでは基本ファイルが端末に保存されます。
  4. ホーム画面に追加しておくと、外でもアプリのように開けます。

Bookkeeping grade 3

簿記3級の基本を仕訳から読む

01

仕訳

取引を借方と貸方に分けて記録します。左右の金額は必ず一致します。

02

現金、預金

現金、普通預金、当座預金など、支払いに使える資産を扱います。

03

売上、仕入

商品を売ると売上、商品を買うと仕入です。商売の中心になる勘定です。

04

費用、収益

収益は利益を増やし、費用は利益を減らします。P/Lの材料になります。

05

売掛金、買掛金

後で受け取る代金が売掛金、後で支払う代金が買掛金です。

06

決算

期末に残高を整理し、利益を計算して、財産の状態をまとめます。

07

P/LとB/S

収益と費用はP/Lへ、資産・負債・純資産はB/Sへ流れます。

決算でやること

期末整理
1. 残高を集める

現金、預金、売掛金、買掛金、売上、仕入、費用などの残高を確認します。

2. 決算整理をする

未払費用、前払費用、減価償却、売上原価など、期末の調整を入れます。

3. P/LとB/Sを作る

収益と費用で利益を出し、資産・負債・純資産で財産の状態をまとめます。

Core map

3表のつながり

P/L

売上から費用を引いて、利益を見ます。

基本式
売上 - 費用 = 利益
見るポイント
粗利率、営業利益率、最終利益

B/S

資産、負債、純資産の状態を見ます。

基本式
資産 = 負債 + 純資産
見るポイント
現金、売掛金、借入金、自己資本比率

C/F

現金がどこで増減したかを見ます。

基本式
営業CF + 投資CF + 財務CF = 現金増減
見るポイント
営業CF、フリーCF、期末現金

C/F 現金の増減を説明する

最初に見る順番

  1. P/Lで利益が出ているかを見る。
  2. C/Fで利益が現金になっているかを見る。
  3. B/Sで資金繰りと借入の重さを見る。

よくある落とし穴

売上が伸びても、売掛金や在庫が増えると現金は残りにくくなります。黒字倒産を避けるにはC/FとB/Sを合わせて見ます。
